### Task 3: Update API Routes for Async Params (Next.js 15+ Breaking Change)

<read_first>
- src/app/api/projects/[projectId]/route.ts (example route with params)
- src/app/api/projects/[projectId]/requests/[requestId]/route.ts (nested params)
- https://nextjs.org/docs/app/building-your-application/upgrading/version-15#params--searchparams (async params breaking change)
</read_first>

<action>
**Critical Breaking Change (Next.js 15+):** Dynamic route params are now async.

1. For ALL routes in `src/app/api/` with dynamic segments `[param]`:

   **Before (Next.js 14):**
   ```typescript
   export async function GET(
     request: NextRequest,
     { params }: { params: { projectId: string } }
   ) {
     const projectId = params.projectId;
   ```

   **After (Next.js 15+):**
   ```typescript
   export async function GET(
     request: NextRequest,
     { params }: { params: Promise<{ projectId: string }> }
   ) {
     const { projectId } = await params;
   ```

2. Apply this pattern to all 13 API route files:
   - `src/app/api/auth/[...nextauth]/route.ts`
   - `src/app/api/projects/[projectId]/route.ts`
   - `src/app/api/projects/[projectId]/board/route.ts`
   - `src/app/api/projects/[projectId]/members/route.ts`
   - `src/app/api/projects/[projectId]/requests/route.ts`
   - `src/app/api/projects/[projectId]/requests/[requestId]/route.ts`
   - `src/app/api/projects/[projectId]/requests/[requestId]/approve/route.ts`
   - `src/app/api/projects/[projectId]/requests/[requestId]/reject/route.ts`
   - `src/app/api/projects/[projectId]/requests/[requestId]/attachments/route.ts`
   - `src/app/api/projects/[projectId]/requests/[requestId]/comments/route.ts`

3. For routes with multiple params (e.g., `[projectId]` and `[requestId]`):
   ```typescript
   export async function GET(
     request: NextRequest,
     { params }: { params: Promise<{ projectId: string; requestId: string }> }
   ) {
     const { projectId, requestId } = await params;
   ```
</action>

<acceptance_criteria>
- All files in `src/app/api/**/route.ts` with `[param]` use `Promise<{` in params type
- All files in `src/app/api/**/route.ts` with `[param]` use `await params` to destructure
- `npx tsc --noEmit` exits 0
- `grep -r "await params" src/app/api/` returns matches for all dynamic routes
</acceptance_criteria>
